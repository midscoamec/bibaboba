How to submit

Как оформить

1
Track title
Название
Format: Artist - Track name
Формат: Артист - Название
2
Files
Файлы
Cover without compression + audio file (MP3 preferred). Video clip - MP4.
Обложка без сжатия + аудиофайл (желательно MP3). Клип - MP4.
3
Genre & hashtags
Жанр и хештеги
List the genre and relevant hashtags.
Укажи жанр и хештеги для релиза.
4
Platforms
Платформы
Where to release? Use the format STY. e.g. Telegram only → T
Куда выкладывать? Формат STY. Например, только Telegram → T
S - Soundcloud T - Telegram Y - Youtube
5
Description optional
Описание необязательно
Any text to accompany the release.
Желаемое описание для релиза.
